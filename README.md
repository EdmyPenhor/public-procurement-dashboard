# Advanced Excel Public Procurement & Supplier Risk Dashboard

## Project Overview

This project analyzes public supplier payments made by Bristol City Council using Advanced Excel. The goal is to transform raw public procurement payment data into an interactive dashboard that helps monitor supplier spending, payment patterns, supplier concentration, high-value transactions and potential procurement risk indicators.

The project was developed as a portfolio project to demonstrate advanced Excel skills, including Power Query, Power Pivot, data modelling, DAX measures, Pivot Tables, Pivot Charts, data quality checks and dashboard design.

## Business Problem

Public institutions need to monitor supplier payments to improve transparency, financial control and procurement decision-making.

This project addresses the following business question:

> How can supplier payment data be transformed into an interactive Excel dashboard to monitor spending patterns, supplier concentration, payment behaviour and potential procurement risks?

## Data Source

The dataset used in this project comes from public supplier payment records published by Bristol City Council / UK Government Open Data.

- **Dataset:** Payments to suppliers with a value over £500 from Bristol City Council
- **Source:** UK Government Open Data / Bristol City Council
- **Period analyzed:** 2024
- **Format:** CSV files
- **Unit of analysis:** Supplier payment transaction

The original raw data is not included in this repository. It can be downloaded from the official UK Government Open Data portal.
